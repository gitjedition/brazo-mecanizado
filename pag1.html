Brazo Mecanizado

Proyecto creado por: Jesus Emmanuel Torres Ramírez 2B

Descripción:

He creado un brazo mecanizado que utiliza un mecanismo de engranes para hacer los giros, un sistema hidráulico para mover la altura y estiramiento del brazo y un interruptor push para poder controlar el agarre magnético. Además, incluye un indicador LED para proporcionar retroalimentación visual.

Este brazo mecanizado está construido principalmente con madera y fue diseñado inicialmente en una libreta de bocetos, fusionando ideas de otros proyectos para llegar a su diseño final.

Este proyecto demuestra la aplicación de conceptos básicos de electrónica, mecánica e hidráulica en la creación de un brazo mecanizado multifuncional y versátil. La combinación de diferentes tecnologías permite realizar una variedad de tareas de forma eficiente y precisa.

Materiales Utilizados:

  • Transistor
  • Engranajes
  • Indicador LED
  • 1 tornillo
  • Alambre magneto
  • Interruptor push
  • Cables
  • Batería o eliminador de 9V
  • Jeringas
  • Mangueras
  • Agua
  • Madera
  • Pegamento industrial
  • Tubos de plástico
